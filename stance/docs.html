Installation

clone the public repository

        >>> git clone https://github.com/RFinkelberg/Stance.git

install the requirements

        >>> pip install -r requirements.txt

download the neural net

        >>> python get_models.py

Quick Start

        >>> cd stance/
        >>> python main.py -v example/squat.mp4

Using Pickle Files

There are three pickle files each containing precomputed skeletons for each frame of the videos.
  • example/squat.mp4
  • example/squatbad.mp4
  • example/squatpoop.mp4
        >>> python main.py -v example/squatbad.mp4 -p
        >>> python main.py -v example/squatbad.mp4 --use_pickle

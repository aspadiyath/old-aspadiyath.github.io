Project Planning

"static diagram of stance system"

Milestone 1 - Setup

template_fit.py
input: two photos of front and profile of user and benchmark zones
output: template skeletons in the benchmark zones
tasks:
  • compute wire skeletons / body ratios of user
  • for each benchmark zone, compute template skeletons using body ratios
etl.py
input: video of user performing motion
output: ordered set of wire skeletons for every frame
tasks:
  • separate video into frames
  • preprocessing on frames to get good results (color correcting, etc.)
  • run OpenPose on every frame
MOTION.py
for any particular motion:
output:
  • benchmark zones
  • skeleton similarity function
  • scoring function

Milestone 2 - Fit

fit.py
input: ordered set of wire skeletons for every frame and similarity function
output: map from template skeleton to user skeleton for every benchmark zone
tasks:
  • use skeleton similarity function to compute similarities for every frame and every benchmark zone
  • find peaks of similarities for every benchmark zone and store in map

Milestone 3 - Score

score.py
input: map from template skeleton to user skeleton for every benchmark zone and scoring function
output: Error score
tasks:
  • combine similarities using scoring function

Milestone 4 - Overlay

overlay.py
input: map from template skeleton to user skeleton for every benchmark zone and scoring function
output: display template skeleton on top of user skeleton
tasks:
  • overlays template skeletons on user's benchmark photos
  • overlays template on user videos
